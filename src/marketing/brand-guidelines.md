# MosqueSteps — Brand Guidelines

## Brand Identity

- **Name:** MosqueSteps (one word, capital M and S)
- **Tagline:** "Turn every step into a blessing"
- **Voice:** Warm, encouraging, authentic, respectful

## Color Palette

| Token | Light Mode | Dark Mode | Usage |
|-------|-----------|-----------|-------|
| Primary (Teal) | hsl(174, 80%, 26%) | hsl(174, 70%, 40%) | Buttons, links, headers |
| Accent (Gold) | hsl(39, 95%, 55%) | hsl(39, 90%, 50%) | Badges, highlights, rewards |
| Background (Cream) | hsl(42, 100%, 97%) | hsl(215, 30%, 8%) | Page backgrounds |
| Foreground | hsl(215, 30%, 15%) | hsl(42, 100%, 95%) | Body text |

## Typography

- **Headings:** Plus Jakarta Sans (700, 800)
- **Body:** Plus Jakarta Sans (400, 500)
- **Arabic text:** Amiri (400, 700)

## Logo Usage

- Always pair the icon with "MosqueSteps" text
- Minimum size: 24px icon
- Always maintain spacing between icon and text

## Islamic Content Standards

- All hadiths must include source reference (book + number)
- Link to sunnah.com for verification when possible
- Use ﷺ after mentioning the Prophet
- Respect scholarly consensus — avoid controversial topics
- Use "Allah" rather than "God" in Islamic contexts

## Accessibility

- Maintain WCAG AA contrast ratios
- All images must have alt text
- Support both light and dark modes
- Ensure keyboard navigation works throughout
